# Editorix

Editorix is an image editing app that is based on matrices and their operations. This is done using the following Python modules: 
1. numpy
2. matplotlib
3. Pillow
4. scipy

## Editing Options

#### 0. Display Image

#### 1. Rotate
    1. Rotate Left
    2. Rotate Right
    3. Rotate by Angle

#### 2. Laterally Invert

#### 3. Apply Filter
    1. Negative Image
    2. Grayscale Image
    3. Red Channel
    4. Green Channel
    5. Blue Channel

#### 4. Pad Image 

#### 5. Mask Image 

#### 6. Trim Image

#### 7. Reduce Colour Intensity

#### 8. Binarize Image

#### 9. Paste Image with Slice

#### 10. Blend Two Images

#### 11. Display Image Details

#### 12. Plot Histogram for Pixel Intensity

#### -1. Quit
